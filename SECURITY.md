The following versions of this project are currently being supported with security updates.

| Version | Supported          |

## Reporting a Vulnerability

I take the security of this project seriously. If you find a security vulnerability, please do NOT open an issue. Instead, report it privately.

Please report security vulnerabilities via email to [abdheshshah111@gmail.com](mailto:abdheshshah111@gmail.com).

I will acknowledge your report within 48 hours and provide a timeline for a fix, if applicable.

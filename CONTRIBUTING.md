# Contributing to Cyberpunk Portfolio

First off, thank you for considering contributing to this project! It's people like you that make it a great tool for everyone.

## How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** issue template.
- Describe the bug in detail and provide steps to reproduce.

### Suggesting Features
- Use the **Feature Request** issue template.
- Explain why the feature would be useful and how it should work.

### Pull Requests
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/amazing-feature`).
3. Commit your changes (`git commit -m 'Add amazing feature'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request using the provided template.

## Style Guidelines
- Use TypeScript for both Frontend and Backend.
- Follow the existing project structure.
- Ensure premium aesthetics are maintained in the UI.

## Code of Conduct
Please note that this project is released with a [Contributor Code of Conduct](./CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
